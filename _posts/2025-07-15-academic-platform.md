## Goal
Build an intuitive, secure, and efficient platform for digital academic management.

## Description
A complete web-based solution for managing academic activities, including students, professors, grades, and majors.

### Key Features
- **Full Management**: CRUD operations for all academic entities.
- **Architecture**: Built on **MVC pattern** for clean code and maintainability.
- **Backend**: Powered by **PHP 8.3** and **MySQL 9.1**.
- **Frontend**: Responsive design using **HTML5, CSS3, JavaScript, and Bootstrap 5**.

### Technologies
- PHP 8.3
- MySQL
- Bootstrap 5
- JavaScript
